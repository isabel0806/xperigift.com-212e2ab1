# Xperigift

A done-with-you gift card revenue consulting service for US-based SMBs ($150k–$2M).

**Primary goal of the website:** Book a free 30-minute Gift Card Revenue Audit.

**Verticals:** Spas, Salons, Restaurants, Golf clubs, Specialty retail, Gun shops.

**Hierarchy (NEVER reverse):**
1. Revenue engine (core offer)
2. Done-with-you service (main value)
3. Infrastructure — TheGiftCardCafe (handled, not surfaced as a separate product)
4. Dashboard (supporting visibility layer only — never the product)

**Strategic positioning:** NOT a SaaS. NOT a competitor to GoHighLevel or marketing platforms. Avoid feature grids, "all-in-one", and software comparisons. Message: "You don't need another tool. You need a system that drives revenue."
